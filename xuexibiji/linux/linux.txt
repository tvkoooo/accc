有这些大概
ls -trl   查看当前目录的文件夹和文件
tail 检测一个文件的附加写入
```
tail -f 文件名 | grep  关键字（比如： id号，包体关键字）
tail -f 2017-12-22.log | grep -E 'test_io_robot_comein_Success|robot_talk_model'        或者
tail -f 2017-12-22.log | grep 'test_io_robot_comein_Success' | grep 'robot_talk_model'  且
```
top  查看进程
grep 这个一般复合用，用于筛选字符串
```
    ls -trl | grep "proj"///查找带proj整行东西
```
ps 查看当前运行的程序有哪些
```
ps -ef  当前运行所有进程罗列出来
ps -ef | grep "redis-server"查找带redis-server整行进程信息
```
	ps -ef | grep lizhi_4
```
cd 目录切换
cp 拷贝
ctrl c 一般都能退出
vim 打开某文件，是个编辑器
```
	vim .bash_profile
	i          插入模式
	Esc        退出插入模式
    dd         删除光标所在行
    dw         删除一个字(word)
	x     删除当前字符
	X     删除前一个字符
	shift+:    出现冒号控制台	
    :wq     保存修改并退出
	:q!     强制退出，放弃修改
```
make 编译工程
cat 将文件输出
mv 移动文件
su 登录root账号
sudo 使用root权限执行某命令
su 账号名 登录某账号
mkdir         创建目录
rm 移除,删除
du                  查看目录大小 du -h /home带有单位显示目录信息
df                  查看磁盘大小 df -h 带有单位显示磁盘信息
ipconfig          查看网络情况
ping                测试网络连通
netstat          显示网络状态信息
```
	netstat -anp | grep 8888 //查看端口占用
```
kill                 杀死进程，可以先用ps 或 top命令查看进程的id，然后再用kill命令杀死进程。
```
	kill -2 pid //退出进程
	kill -9 pid //强行结束进程
```
man                命令不会用了，找男人  如：man ls
tar 解压缩
```
    tar -cvf /home/abc.tar /home/abc              只打包，不压缩
    tar -zcvf /home/abc.tar.gz /home/abc        打包，并用gzip压缩
    tar -jcvf /home/abc.tar.bz2 /home/abc      打包，并用bzip2压缩
```
